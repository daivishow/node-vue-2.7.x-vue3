🚀 Instalando Vue.js

Método 1: Via CDN (Mais Simples)

A forma mais rápida para começar é usar o Vue.js via CDN. Copie o código abaixo em seu arquivo HTML:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Hello Vue</title>
</head>
<body>
    <div id="app">
        <p>{{ message }}</p>
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue@2.7.14/dist/vue.js"></script>
    <script>
        new Vue({
            el: '#app',
            data: {
                message: 'Hello Vue.js!'
            }
        });
    </script>
</body>
</html>

Método 2: Via NPM (Recomendado para Projetos Maiores)

Se você tem Node.js instalado, use o npm para instalar Vue:

npm install vue@2.7.14

Em seu arquivo JavaScript:

import Vue from 'vue'

new Vue({
    el: '#app',
    data: {
        message: 'Hello Vue.js!'
    }
})

Conceitos Fundamentais

Data (Dados)

O objeto data contém os dados reativos da aplicação:

data: {
    message: 'Hello',
    count: 0,
    user: { name: 'João', age: 25 }
}

Interpolação {{ }}

Use chaves duplas para exibir dados no HTML:

<p>{{ message }}</p>
<p>{{ count + 1 }}</p>
<p>{{ user.name }}</p>

Instância Vue

A instância Vue é criada com new Vue() e requer:

  • el: Elemento DOM onde Vue vai ser montado
  • data: Dados reativos da aplicação
  • methods: Funções que manipulam os dados

Exemplo Prático: Contador

Contador: {{ count }}

Este é seu primeiro aplicativo Vue.js funcionando! 🎉

💡 Dica: Você pode inspecionar a página com F12 e ver os dados reativos. Abra o console e digite app.count = 100 para ver a mágica acontecer!

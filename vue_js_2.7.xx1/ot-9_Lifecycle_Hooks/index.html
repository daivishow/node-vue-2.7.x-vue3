🔄 Lifecycle Hooks - Ciclo de Vida do Vue

Fases do Ciclo de Vida

1️⃣ Creation
├─ beforeCreate: Dados ainda não estão prontos
└─ created: ✅ Dados iniciados, pronto para usar

2️⃣ Mounting (Renderização)
├─ beforeMount: Template compilado, DOM ainda não atualizado
└─ mounted: ✅ Componente renderizado no DOM

3️⃣ Updating (Mudanças)
├─ beforeUpdate: Dados mudaram, DOM ainda não
└─ updated: ✅ DOM atualizado

4️⃣ Destruction
├─ beforeDestroy: Antes de ser destruído
└─ destroyed: ✅ Componente removido

📋 Hooks Principais e Seus Usos

1. beforeCreate()
Executado antes de initializar as propriedades do componente. data e methods ainda não estão disponíveis.
2. created()
✅ Ideal para: Buscar dados da API, inicializar variáveis, fazer requisições HTTP
3. beforeMount()
Executado antes do componente ser inserido no DOM. Template foi compilado.
4. mounted()
✅ Ideal para: Acessar o DOM, usar bibliotecas jQuery, configurar plugins
5. beforeUpdate()
Executado antes dos dados serem atualizados no DOM.
6. updated()
✅ Ideal para: Reagir a mudanças de dados, atualizar gráficos
7. beforeDestroy()
Executado antes do componente ser removido. Ainda tem acesso aos dados.
8. destroyed()
✅ Ideal para: Limpar listeners, cancelar requisições, limpar timers

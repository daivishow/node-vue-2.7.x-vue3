🧩 Componentes Reutilizáveis

Componente: CustomButton

Botão Primário Botão Sucesso Botão Perigo
Props: type (primary, success, danger)
Eventos: @click

Componente: Card com Slots

Este é um componente card reutilizável com slots para conteúdo customizável.
  • Props - Passam dados pai → filho Comunicação
  • Emit - Eventos filho → pai Comunicação
  • Slots - Conteúdo customizável Flexibilidade
Props: title
Slots: default (conteúdo)

Componente: Alert

ℹ️ Isto é uma mensagem informativa ✅ Operação realizada com sucesso! ⚠️ Cuidado com esta ação! ❌ Erro ao processar a requisição!
Props: type (info, success, warning, danger)
Slots: default (conteúdo)

Componente: Contact List

Props: contact (objeto com id, name, email)
Eventos: @delete

📋 Conceitos de Componentes

Props: Dados passados de pai para filho

<my-component :title="parentData"></my-component>

// No componente
props: ['title']

Emit: Eventos do filho para pai

<my-component @customEvent="handleEvent"></my-component>

// No componente
this.$emit('customEvent', data)

Slots: Conteúdo customizável

<my-card>
    <p>Este conteúdo vai para o slot</p>
</my-card>

// No componente
<div><slot></slot></div>
